FAQ

What are these starters I keep seeing?

Each starter represents an empty template to begin with, having the components in the name present.

How do I install the item?

Copy and paste all the items in the code folder to your server to your desired location. 

What are those i class "ion " things?

That is Font Awesome, the amazing icons we used for this item you can learn more about them and find a cheatsheet on how to use them here http://ionicons.com/

How do I change the contact form information?

There is only one PHP file in the item, please open that item and replace the email address with yours. 

How do I make the login page / sign in page / calendar page / news / store / blog  to work? 

Please remember that this is a site template and is not connected to any CMS or any databases, so these features are mearly template designs, they are HTML and CSS and need to be connected to a back-end to work. 

 What is the Mobile UI section?

The mobile UI section is comprised of elements that you can use to create an app. Please note that converting the item to an app is entirely up to you. If these elements don't suit you and you wish to reduce the page load, simply open framework.css and scroll to the very bottom. You can remove them from there. 

I want to add my own JS, it's not working, why?

Add any JS you want starting from line 4. THis is very important since the AJAX refreshes the functions and they are present starting with line 3 and down to 927. When you tap a link that entire zone is refreshed. Be sure to import your script in all pages you want it to work. My suggestion is to copy the min version of your script in plugins.js and import it there as we imported ours. 

This uses AJAX? How to I use a simple link? They don't work! 

Any link that points outside the main item should have the class default-link. This will terminate the smooth state / ajax page loading for that link and it will work like anyother regular link! 

I don't want AJAX. I want the old school loading style. How? 

That's actually very easy. Open custom.js and delete lines 927 to 993

I'm a tech guru. I want more API's for the AJAX. I need them. 

Everything you need to know to customize the AJAX and more API's can be located on the plugin page over at https://github.com/miguel-perez/smoothState.js

I want to build an app using PhoneGap and Cordova. Can you help me? 

We only provide support for the item as is, meaning a site template. PhoneGap and Cordova's pages state clearly that they work wiht ANY html, css and JS item on the planet. So compatiiblity is not an issue. Furthermore, we use the latest and easiest code techniques to assure there won't be issues. 

Do I need an extended license?

If your app will be paid or have in-app paid features, yes. You do. We will pursue legal action against any app on any application store that uses our code and can't provide a license for an extended license use.

The menu is comprised of a main class footer-menu and a secondary class that is footer-menu-scroll that will make the menu show or hide depending on it's position on the screen. If you want it to be permanently visible, replace footer-menu-scroll with footer-menu fixed.

Inside the footer there are anchor classes. Inside each there is an icon, and a text.

Notice that the last 3 anchors have the class tablet-link. These 3 icons will only show on tablets.
